# HexSOC AI Enterprise Roadmap

HexSOC AI is being built as an enterprise cybersecurity technology platform, not a student project. This roadmap tracks completed platform capabilities and planned phases.

## Completed

- RBAC authentication
- Admin user management
- Collector API keys
- Real-time WebSocket dashboard updates
- Persistent agent loop
- Offline queue and retry
- Dead-letter telemetry queue
- Environment-aware agent configuration
- Agent secret masking
- Windows scheduled service installer
- Agent state and duplicate prevention
- Live collector health monitoring
- Detection engine
- Threat intelligence enrichment
- MITRE ATT&CK mapping
- Correlation and attack-chain engine
- Graph investigation view
- AI Analyst Copilot
- Case management and analyst notes
- SOC report export

## Next

- Phase 3C.1 Real Windows Event Log Collector
- Phase 3C.2 Sysmon XML parsing and normalization
- Phase 3C.3 Sigma-style detection pipeline
- Phase 3C.4 Fleet management
- Phase 3C.5 Remote policy/config management
- Phase 3D Multi-tenant enterprise architecture
- Phase 4 AI anomaly detection and investigation intelligence

## Product Direction

HexSOC AI should continue moving toward an enterprise SOC/XDR architecture:

- Reliable endpoint telemetry collection
- Managed collector fleet visibility
- Policy-driven remote configuration
- Detection-as-code
- MITRE-aligned investigation workflows
- Analyst case management
- AI-assisted triage and reasoning
- Production-safe deployment and operations
